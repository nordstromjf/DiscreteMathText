Exercise 5.

Define the function \(F:\mathbb{Z}\times\mathbb{Z}\rightarrow \mathbb{Z}\times\mathbb{Z}\) where for all ordered pairs \((a, b)\) of integers, \(F(a, b)=(2a+1, 3b-2)\text{.}\)
  1. Find \(F(4, 4)\text{.}\)
  2. Find \(F(2, 1)\text{.}\)
  3. Find \(F(3, 2)\text{.}\)
  4. Find \(F(1, 5)\text{.}\)
