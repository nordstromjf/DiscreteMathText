Reading Question 5.

    Let \(D=\mathbb{Z}\text{.}\) Let \(P(x)\) be “\(x^2\) is even.”
    True or False: \(\sim(\forall x\in D, P(x)).\)

  • True.

  • This is the negation of \(\forall x\in D, P(x)\text{,}\) which was false, so it must be true.

  • False.

  • This is the negation of \(\forall x\in D, P(x)\text{,}\) which was false, so it must be true.
