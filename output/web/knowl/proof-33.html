Proof

\begin{align*} A-(A\cap B) &= A\cap(A\cap B)^C \ \ \text{ Set Difference Law}\\ &= A\cap(A^C\cup B^C) \ \ \text{ DeMorgan's Law}\\ &= (A\cap A^C)\cup (A\cap B^C) \ \ \text{ Distributive Law}\\ &= \emptyset\cup (A\cap B^C) \ \ \text{ Complement Law}\\ &= A\cap B^C \ \ \text{ Identity Law}\\ &= A-B \ \ \text{ Set Difference Law} \end{align*}
