If we think about the first row as actually being the 0th row, we can make a triangle with the binomial coefficients:
\(\binom{0}{0}\)
\(\binom{1}{0}\) \(\binom{1}{1}\)
\(\binom{2}{0}\) \(\binom{2}{1}\) \(\binom{2}{2}\)
\(\binom{3}{0}\) \(\binom{3}{1}\) \(\binom{3}{2}\) \(\binom{3}{3}\)
