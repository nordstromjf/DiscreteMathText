Proof by Cases.

To prove \(\forall x\in D, P(x)\) by cases,
  1. Break \(D\) into smaller subsets, \(D_1, D_2, \ldots D_k\text{.}\) Make sure every element in \(D\) is in one of the subsets.
  2. Prove each case separately. Make sure you state each case clearly.
    Case 1: \(x\in D_1\)
    Proof of \(P(x)\text{.}\)
    Case 2: \(x\in D_2\)
    Proof of \(P(x)\text{.}\)
    etc.
  3. Conclude \(P(x)\) holds for all \(x\in D\text{.}\)
