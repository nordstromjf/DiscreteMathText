Exercise 2.

Let \(X=\{a, b, c\}\) and \({\cal P}(X)\) be the power set of \(X\text{.}\) Define the relation \(R\) on \({\cal P}(X)\) by
\begin{equation*} A \ R\ B \iff A \text{ has the same number of elements as } B. \end{equation*}
  1. Is \(\{a, b\}\ R\ \{b, c\}\text{?}\)
  2. Is \(\{a\}\ R\ \{a, b\}\text{?}\)
  3. Is \(\{c\}\ R\ \{b\}\text{?}\)
